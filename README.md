# 유병자 인수예외질환 검색

보험대리점 설계사용 내부 검색 사이트 초안입니다.

## 목표
- 보험사 간편보험 예외질환 데이터를 빠르게 검색
- 질환코드 / 질환명 / 별칭 기준 검색 지원
- 보험사 / 가능상품구분 필터 지원
- 내부 관리자 코드 기반 접근 제어
- 검색 로그 저장
- 추후 OpenAI 기반 자연어 검색 확장

## 예정 기술 스택
- Frontend: HTML, CSS, JavaScript
- Hosting: Netlify
- Backend: Netlify Functions
- Database: Supabase
- AI: OpenAI API
